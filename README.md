# Rubatar

Generates random 5x5 avatars.

## Installation

Add this line to your application's Gemfile:

    gem 'rubatar'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rubatar

## Usage

    rubatar --size 128x128 --output avatar.png
